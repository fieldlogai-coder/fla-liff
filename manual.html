1はじめに

FIELDLOGはLINEの友だちとして使います。アプリのインストールは不要です。

  1. 会社からもらったQRコードで友だち追加
  2. 名前の登録を求められたら、画面の案内に沿って入力（管理者が登録済みの場合は不要）
  3. トーク画面の下に出るメニューから操作します

メニューが出ていないときは、入力欄の横のメニューアイコンをタップすると開きます。

2日報を出す（毎日）

メニューの「日報報告」をタップして、あとは出てくるボタンを選ぶだけです。文字入力はほぼありません。

  1. 種類：いつも通りなら「通常報告」。前日と同じ内容なら「昨日と同じ」で一発登録
  2. 現場：ボタンから選ぶ（新しい現場は「＋新しい現場」から追加できます）
  3. 勤務区分：終日 or 半日（半日は理由を選びます）
  4. 異常の有無：「異常なし」or「異常あり」→ あれば内容を文字で送信
  5. 引き継ぎ：あれば内容を送信、なければ「なし」
  6. 最後に「完了」。同じ日に別の現場でも働いたら「別の現場を追加」
🕐 目安は1分。「異常あり」で報告した内容は、自動で現場メモにも転記されて職長・社長に共有されます。

3休みの日

メニューの「休み報告」をタップするだけ。確認メッセージが返ってきたら完了です。

（休み報告 をタップ）
了解しました。お休みを登録しました。お大事に！

同じ日に2回押しても二重登録にはなりません。

4出した報告を確認する・直す

確認

「日報確認」→「今日の集計」または「今月の累計」を選ぶと、登録内容がカードで表示されます。

修正

「日報修正」→ 直したい報告を選ぶ →「現場名」「勤務区分」「異常・引き継ぎ内容」のどれを直すか選んで入力し直します。

修正できるのは当日分です。前日以前の修正は管理者に相談してください。

5現場メモ（写真つき伝言板）

メニューの「現場メモ」をタップすると、現場ごとの掲示板が開きます。

  • 投稿：現場を選んで、メモと写真（任意）を投稿
  • 区分バッジ：メモ → 未対応 → 対応中 → 完了 の順にバッジをタップで切り替え。未対応・対応中は一覧の上に固定されます
  • 確認：他の人の投稿は「確認」ボタンで既読がつきます

6困ったとき

  • 途中でやめたい・ボタンが効かない → ✖ キャンセルをタップ（どの画面にも付いています）。最初に戻ります
  • 入力の途中で放置してしまった → 約1時間で自動的に最初に戻ります。そのまま次の操作をどうぞ
  • 「登録されていません」と言われる → 管理者に「作業員登録」を依頼してください
  • 間違えて送った → 「日報修正」から直せます
1. 準備（最初だけ）マスタ登録の4点セット 2. 毎月の請求現場を選んで発行まで 3. 締め日のしくみ期間は自動で決まる 4. 値引き・端数調整丸い金額にする 5. 発行後の修正・印刷改訂・取消・PDF 6. 作業員・現場の管理「設定」コマンド

1準備（最初だけ・約15分）

メニュー右上の「請求」タブ → 「請求を開く」 → 画面右上の「マスタ」から、台帳を4つ登録します。

  1. 自社情報 — 会社名・代表者・インボイス登録番号・振込口座（請求書に印字されます）
  2. 元請（請求先） — 会社名・締め日・請求書に必須の欄（工事番号など）＋この元請の現場にチェック
  3. 工種と常用単価 — 工種（例：塗装・シール）と元請ごとの終日単価（半日は自動で半額・変更可）※常用請求を使う場合
  4. 請負金額 — 現場×元請の契約金額と工事名 ※出来高請求を使う場合
👤 請求メニューは権限（can_bill）のある人だけに表示されます。担当を増やしたいときはシステム管理者に依頼してください。

2毎月の請求（現場ごとに1枚）

  1. 請求一覧 →「＋ 今月の請求をつくる」
  2. 現場を選ぶ（元請と締め期間は自動で決まります）
  3. 種類を選ぶ：
    • 出来高（請負）…今月分の金額（税抜）を入力。進捗バー・累計・残額は自動表示
    • 常用（人工）…工種を選ぶ → 終日/半日の人工数（単価は自動で入ります）→ 材料費（あれば）。複数の工種は「＋この工種を明細に追加」で1枚にまとめられます
  4. 確認画面：明細・金額・必須欄（工事番号など）をチェック。値引きはここで（→ 4章）
  5. 「発行する」 → 請求書番号が自動で付いて確定

3締め日のしくみ

請求の期間は、元請マスタの締め日から自動で決まります。対象になるのは「直近に締めた期間」です。

  • 25日締めの例：8/26〜9/25 の間に作ると「8/25締め分（7/26〜8/25）」の請求書になります
  • 末日締めの例：9月中に作ると「8月分（8/1〜8/31）」＝月初に前月分を請求する流れです
⚠️ 同じ現場・同じ締め期間の請求書は2枚作れません（二重請求の防止）。作り直すときは先に「取消」してください。

4値引き・端数調整

確認画面の「端数調整（値引き）」を使うと、請求額を丸い金額にできます。

  1. 今の税抜合計が表示される（例：¥533,400）
  2. 丸めたい金額を入れる（万円未満切捨てが最初から入っています。例：¥530,000）
  3. 「値引き行を追加する」→ 差額が「値引き −¥3,400」として明細に載り、請求書にもそのまま印字されます

駐車場代などの追加は「＋その他の調整」から名目と金額で登録できます。

5発行後の修正・取消・印刷

  • 修正する（改訂）：一覧から請求をタップ →「修正する」。番号は変わらず第2版・第3版…として履歴に残ります
  • 取消する：無効化されます（履歴には残る）。同じ期間で作り直せるようになります
  • 印刷・PDF：詳細 →「📄 請求書を表示」→ A4の請求書。PCのブラウザで開いて「印刷」→「PDFに保存」が確実です

6作業員・現場の管理（LINEトークで）

トークで設定と送ると、管理メニューが出ます（管理者のみ）。

  • 作業員を追加/削除 — 名前と役割（作業員・職長・管理者）で登録
  • 現場を追加/終了 — 終了した現場は選択肢から消えます
  • 現場を確認/作業員を確認 — 登録一覧の表示
💡 単価の登録・変更は「請求」タブのマスタ（工種と常用単価）から行います。
